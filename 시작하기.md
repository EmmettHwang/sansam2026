# 🚀 팜랜드 산양산삼 - 빠른 시작 가이드

## ⚠️ 중요: 백엔드 서버 실행 필수!

**"이미지 업로드 실패"** 또는 **"갤러리 로드 실패"** 에러는  
**백엔드 서버(main.py)가 실행되지 않아서 발생합니다!**

---

## 🎯 1단계: 백엔드 서버 실행 (필수!)

### Windows
```bash
서버실행_FastAPI.bat (더블클릭)
```

### Mac / Linux
```bash
chmod +x 서버실행_FastAPI.sh
./서버실행_FastAPI.sh
```

### ✅ 정상 실행 확인

터미널에 다음 메시지가 표시되면 성공:
```
INFO:     Uvicorn running on http://127.0.0.1:8000
INFO:     Application startup complete.
```

---

## 🎯 2단계: 관리자 페이지 접속

브라우저에서 접속:
```
http://localhost:8000/admin
```

### ✅ 정상 접속 확인

- **갤러리 관리** 탭에 5개 카드 표시:
  - 🏞️ 재배지
  - 🌿 산양산삼
  - ⚙️ 선별과정
  - 📦 포장
  - 📄 인허가
  
- **간단구매 상품 관리** 탭 표시

---

## 🎯 3단계: 이미지 업로드 테스트

### 갤러리 이미지 업로드
1. **갤러리 관리** 탭 클릭
2. 원하는 카드 선택 (예: 📄 인허가)
3. 이미지 드래그앤드롭
4. "이미지가 업로드되었습니다" 메시지 확인

### 상품 등록
1. **간단구매 상품 관리** 탭 클릭
2. 상품 정보 입력:
   - 상품명: `치악산 산양산삼 10년근`
   - 가격: `150000`
   - 설명: `치악산에서 10년간 자란 프리미엄 산양산삼`
3. 상품 이미지 드래그앤드롭
4. **💾 저장** 버튼 클릭
5. 상품 목록에서 확인

## 🐛 문제 해결

### ❌ "이미지 업로드 실패"

**원인**: 백엔드 서버(main.py) 실행 안 됨

**해결**:
```bash
서버실행_FastAPI.bat 실행
```

### ❌ "갤러리 로드 실패"

**원인**: 백엔드 서버(main.py) 실행 안 됨

**해결**:
```bash
서버실행_FastAPI.bat 실행
```

### ❌ "갤러리 카드가 안 보임"

**원인**: 백엔드 서버(main.py) 실행 안 됨

**해결**:
```bash
서버실행_FastAPI.bat 실행
```

### ❌ "Cannot connect to database"

**원인**: MySQL 서버 접속 실패

**해결**:
1. MySQL 서버 실행 확인
2. `main.py`의 DB 설정 확인:
   ```python
   DB_CONFIG = {
       'host': 'bitnmeta2.synology.me',
       'port': 3307,
       'user': 'iyrc',
       'password': 'dodan1004~!@',
       'database': 'sansam'
   }
   ```

### ❌ "FTP connection failed"

**원인**: FTP 서버 접속 실패

**해결**:
1. FTP 서버 실행 확인
2. `main.py`의 FTP 설정 확인:
   ```python
   FTP_CONFIG = {
       'host': 'bitnmeta2.synology.me',
       'port': 2121,
       'user': 'ha',
       'password': 'dodan1004~',
       'base_path': '/sansam/'
   }
   ```
