## 🎯 버전 규칙

### 메이저 버전 (1.x.x)
- 주요 기능 변경
- 호환성 깨지는 변경
- 예: 1.0 → 2.0

### 마이너 버전 (x.1.x)
- 새로운 기능 추가
- 기존 기능 개선
- 예: 1.0 → 1.1

### 날짜시간 (x.x.20260120-1530)
- 배포 날짜: YYYYMMDD
- 배포 시간: HHMM
- 예: 20260120-1530 = 2026년 1월 20일 15:30
