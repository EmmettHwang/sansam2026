### 2️⃣ 구매하기 버튼 오류 수정
**문제**: 구매하기 버튼 클릭 시 에러 발생

**원인**: 존재하지 않는 `purchase.php` 페이지로 이동 시도

**해결**: 문의 폼으로 스크롤 + 안내 메시지
```javascript
function goToPurchasePage(productId) {
    // 상품 정보 저장
    sessionStorage.setItem('selectedProductId', productId);
    
    // 문의 폼으로 스크롤
    const contactForm = document.getElementById('contact-form');
    if (contactForm) {
        contactForm.scrollIntoView({ behavior: 'smooth', block: 'center' });
        
        setTimeout(() => {
            alert('해당 상품에 대한 구매 문의를 아래 폼에 작성해주세요.\n\n☎️ 전화: 010-2512-6818\n💬 문자: 010-2512-6818');
        }, 500);
    } else {
        alert('상품 구매 문의\n\n☎️ 전화: 010-2512-6818\n💬 문자: 010-2512-6818\n\n상담 시간: 08:00 ~ 20:00 (연중무휴)');
    }
}
```

### 3️⃣ 간단구매 디자인 대폭 개선

#### 📐 그리드 레이아웃
```css
.product-grid {
    grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
    gap: 32px;  /* 24px → 32px */
    padding: 20px 0;
}
```

#### 🎨 카드 디자인
**개선 사항**:
- 카드 모서리: `12px` → `16px` (더 부드러운 느낌)
- 그림자 효과 개선: `0 2px 8px rgba(0,0,0,0.08)`
- **상단 그라데이션 바** 추가 (호버 시 나타남)
- 호버 효과 강화: `translateY(-6px)` → `translateY(-8px)`
- 테두리 색상: 투명 → 호버 시 Primary 색상

```css
.product-card::before {
    content: '';
    position: absolute;
    top: 0;
    left: 0;
    right: 0;
    height: 4px;
    background: linear-gradient(90deg, var(--primary-color), var(--primary-light));
    transform: scaleX(0);
    transition: transform 0.3s ease;
}

.product-card:hover::before {
    transform: scaleX(1);
}
```

#### 🖼️ 이미지 개선
```css
.product-image {
    height: 280px;  /* 260px → 280px */
    background: linear-gradient(135deg, #f5f7fa 0%, #e8ebe8 100%);
}

.product-card:hover .product-image {
    transform: scale(1.05);  /* 호버 시 확대 효과 */
}
```

#### 💰 가격 표시 개선
```css
.product-price {
    font-size: 28px;  /* 24px → 28px */
    font-weight: 800;  /* 700 → 800 */
    letter-spacing: -0.5px;
}

.product-price::before {
    content: '🌿 ';  /* 아이콘 추가 */
    font-size: 20px;
}
```

**표시 결과**: 🌿 150,000원

#### 🛒 구매 버튼 개선
```css
.product-buy-btn {
    padding: 16px;  /* 14px → 16px */
    border-radius: 12px;  /* 8px → 12px */
    font-size: 17px;  /* 16px → 17px */

    background: linear-gradient(135deg, var(--primary-color) 0%, var(--primary-dark) 100%);
    box-shadow: 0 4px 12px rgba(74, 124, 42, 0.3);
}

.product-buy-btn:hover {
    transform: translateY(-2px);
    box-shadow: 0 6px 20px rgba(74, 124, 42, 0.4);
}
```
