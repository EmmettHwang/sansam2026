# Images Folder

이 폴더에는 산양산삼 랜딩 페이지에 사용될 이미지 파일들이 저장됩니다.

## 필수 이미지 5개

AI Drive를 통해 아래 파일들을 업로드해 주세요:

1. **hero-bg.jpg** (1920x1080px) - 히어로 섹션 배경 이미지
2. **farm-location.jpg** (800x600px) - 재배지 전경 사진
3. **ginseng-detail.jpg** (800x600px) - 산양산삼 근접 사진
4. **processing.jpg** (800x600px) - 선별/포장 과정 사진
5. **packaging.jpg** (800x600px) - 포장 완성 샘플 사진

## 업로드 방법

1. AI Drive를 엽니다
2. `/chamland-ginseng/images/` 경로로 이동합니다
3. 위 5개 파일을 드래그&드롭으로 업로드합니다

## 참고사항

- 이미지가 없으면 자동으로 회색 placeholder가 표시됩니다
- 파일명은 정확하게 맞춰주세요 (대소문자 구분)
- 이미지 용량은 각 200KB 이하 권장합니다
